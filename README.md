# Getting Started with the application

### `First Step`

Clone the repo

### `Second Step: Run the Backend`

- Open the `backend` folder with a terminal and enter this command: `npm install`
- After the installation is finished, run the backend with the command: `npm start`

### `Third Step: Run the Frontend`

- Open the `frontend` folder with a terminal and enter this command: `npm install`
- After the installation is finished, run the frontend with the command: `npm start`
